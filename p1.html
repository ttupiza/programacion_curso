Contenido Programático - Curso Front-End (HTML, CSS, JavaScript)

Módulo Tema Subtemas Duración Estimada
Módulo 1: Introducción a la Web y HTML Fundamentos de la Web ¿Qué es Internet?, ¿Qué es la World Wide Web?, Funcionamiento Cliente-Servidor, Navegadores Web, Herramientas de Desarrollo. 2 horas
Introducción a HTML Estructura básica de un documento HTML, Etiquetas fundamentales (headings, párrafos, enlaces, imágenes), Atributos globales y específicos. 3 horas
Elementos de Bloque y Línea Diferencias y usos, Elementos semánticos (header, nav, main, article, aside, footer), Listas (ordenadas y desordenadas), Tablas. 4 horas
Formularios Elementos de formulario (input, textarea, select, button), Atributos de los formularios, Validación básica en HTML5. 4 horas
Módulo 2: Estilando con CSS Introducción a CSS ¿Qué es CSS?, Sintaxis de CSS (selectores, propiedades, valores), Formas de aplicar CSS (en línea, interno, externo), Especificidad y Cascada. 3 horas
Selectores CSS Selectores básicos (etiqueta, clase, ID), Selectores de atributo, Pseudo-clases y Pseudo-elementos, Selectores combinadores. 4 horas
Modelo de Caja (Box Model) Contenido, padding, border, margin, Dimensiones, Box-sizing. 3 horas
Layouts con CSS Display (block, inline, inline-block, flex, grid), Posicionamiento (static, relative, absolute, fixed, sticky), Float y Clear. 5 horas
Módulo 3: Interactividad con JavaScript Fundamentos de JavaScript ¿Qué es JavaScript?, Sintaxis básica (variables, tipos de datos, operadores), Estructuras de control (condicionales, bucles), Funciones. 4 horas
DOM (Document Object Model) ¿Qué es el DOM?, Selección de elementos (getElementById, getElementsByClassName, querySelector, querySelectorAll), Modificación de elementos (texto, atributos, clases, estilos). 5 horas
